# Privacy

KeyHub Desktop is fully local by default.

- No account is required.
- No analytics or telemetry is collected.
- No secret, process, path, server, or usage data is uploaded by KeyHub.
- Network access occurs only when the user explicitly tests or deploys to a configured SSH server.
- Clipboard content is excluded from Windows clipboard history and cloud clipboard where supported, and can be cleared automatically.
- Import scanning reads only the path selected by the user and never deletes source files automatically.
